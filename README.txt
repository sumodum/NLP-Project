Ensure python version is at least 3.10!

MAC OS:

Download the repository and locate it in terminal.
Install virtualenv using sudo pip install virtualenv
Create virtual environment using python3.10 -m venv "myenv" 
Start it using source myenv/bin/activate
Download required libraries and packages using pip install -r requirements.txt

To run Part1:
python3 Question1.py

To run Part2:
ipython
run Question2.ipynb

To stop, type deactivate in terminal to exit the environment.
WINDOWS:

Download the repository and locate it in terminal.
Install virtualenv using pip install venv
Create virtual environment using python3.10 -m venv "myenv" then start it using .\\myenv\Scripts\activate
Download required libraries and packages using pip install -r requirements.txt
